## 0 · 一眼看懂（30 秒讲清这游戏）

**一句话定义：**
《符塔》是一款玄幻「画符塔防 Roguelite」微信小游戏。你是一名修士，**带 5 张符出战**布成法阵，守住中央的**护道阵眼**；靠**相生摆位、克制对位、可选手绘画符**，在 **1–3 分钟**一局的波次防守里越级斩妖。局外**无限提升修为境界**、解锁系统、爬无尽符塔。

**面向谁：** 微信生态的轻度~中度玩家。手残/摸鱼能全自动推图，硬核能靠脑子和手翻盘。

**一局只做三件事（核心爽点三件套）：**

| 三件套 | 操作 | 收益 |
|---|---|---|
| **摆位** | 把符放进与它「相生」的区域 | 伤害 ×1.3 |
| **对位** | 用「克制」怪物属性的符去打 | 伤害 ×1.5 |
| **画功** | 手绘画得好（可选） | 再 +20% / 加隐藏特效 |

> 设计哲学一句话：**战力打底（下限），技巧翻盘（上限）。** 同一张符，靠这三件套输出能差到 3 倍（第五部分有完整算例）。

**类型定位：自动战斗为主 + 可选手绘动作层。** 刻意避开「塔防 + 挂机」红海——别人挂机，你动脑 + 动手；但又不逼手残硬操作。

**为什么会留人（留存逻辑）：**
- 短期：每局随机阵位 → 每局最优解不同，不腻。
- 中期：下一个**突破境界**是明确目标，解锁新系统。
- 长期：**等级不封顶** + 无尽符塔排名 + 五行专精圆满，永远有东西追。

**★新增：v3 审阅结论（一眼版）**

| 项 | 结论 | 必做动作 |
|---|---|---|
| 核心玩法 | 成立，「相生摆位 + 克制对位 + 画符」有差异化。 | 先做 20 关首测，不再继续堆系统。 |
| 当前最大风险 | 部分内容仍是“方向/示例”，研发缺少可直接配置的数。 | 第六部分补足战斗参数、20 关、怪物模板、符模板、升级消耗。 |
| 最需修复 | 3 次复活偏重、道行点命名混淆、PvP 圆满克制过强、法器“仅氪”表述偏 P2W。 | 在 1/2/3/4/8 章分别修正，并用 `★修正` 标出。 |
| 首测版本边界 | 20 个主线关 + 1 个材料本 + 1 个突破本 + 10 张基础符 + 8 类怪。 | PvP、宗门、跨服、灵兽、灵宝先不进首测。 |
| 验收标准 | 玩家能从 1-1 打到 1-20，卡关后知道去哪补战力。 | 1-1~1-3 通过率 ≥90%，1-20 通过率 25%~40%。 |

# 第一部分 · 你能做哪些操作（操作清单）

## 1·A 局内操作（一局 1–3 分钟之内）

| 阶段 | 操作 | 怎么做 | 作用 |
|---|---|---|---|
| ① 准备（15s） | **拖拽布阵** | 在本局随机开放的 6 个区域格里放入 5 张符 | 凑相生 ×1.3，并取舍弃掉哪个格 |
| ① 准备 | **查看阵位五行** | 点区域看本局属性 + 相生/错配提示 | 决定怎么摆 |
| ② 战斗（90–120s） | **自动 / 点击施放** | 默认自动放符；也可点符手动放 | 输出 |
| ② 战斗 | **凌空画符**（可选） | 手指描画符文笔迹 | 吻合度越高加成越高（详见 2.7） |
| ② 战斗 | **召出成符** | 一键放出局外预存的「成符」 | 手残的保底强招 |
| ② 战斗 | **用灵机点** | 第 2/4/6 波各得 1 点，二选一：升级符 or 融合 | 局内即时变强 |
| ③ 结算 | **领奖 / 扫荡 / 复活** | 通关领奖；已通关关卡可扫荡；阵眼破可复活 | — |

> **★修正：命名统一。** 局内临时资源统一叫**灵机点**；局外升级经验继续叫**道行经验**。不再混用“道行点”，避免玩家和配置字段混淆。

## 1·B 局外操作（养成 / 系统）

| 系统 | 操作 | 作用 |
|---|---|---|
| **出战配置** | 选定 5 张符 + 各自五行 | 决定本局 build（深度不靠加槽，靠换带哪 5 个） |
| **修为境界** | 攒道行经验升「层」、满级「突破/渡劫」 | 涨等级、解功能、长基础三围（**不封顶**） |
| **五行修为** | 把属性点分配到金木水火土五条线 | 长攻击、走克制专精、冲「圆满」 |
| **装备养成** | 法器/道袍/首饰/灵兽/灵宝/形象 强化与更换 | 堆战力（差异化主力） |
| **融合** | 用灵机点 + 五行精解锁/强化组合技 | 解锁雷/冰/毒/幽/阴/阳等合成元素技 |
| **副本** | 刷材料本 | 产养成材料 |
| **扫荡** | 一键扫已通关关卡 | 省时拿产出 |
| **PvP 段位赛** | 异步排位对战 | 段位、屠榜 |
| **无尽秘境（符塔）** | 一层层往上爬 | 无限挑战，消化高境界，冲周榜 |

## 1·C 新手首通路径（FTUE，前 3 分钟体验）

1. 开局直接进 **1-1**，系统**预置一套相生阵**让你"赢一把"建立爽感。
2. 第 2 波弹出**灵机点**提示 → 教"升级 vs 融合"。
3. 1-1 通关 → 解锁"自己摆阵"，1-2 起阵位随机。
4. 练气 3 层解锁**画符进阶档**（先送一次手绘教学，画好给即时正反馈）。
> 原则：**先赢一把再教系统**，系统跟着境界一个个开，绝不一上来糊脸 10 个入口。

# 第二部分 · 核心玩法（落地）

## 2.1 五行两个循环（v2 缺失，先钉死）★

整套战斗都建立在这两个循环上，必须先写清：

- **相生（区域 ↔ 符，决定摆位）**：木→火→土→金→水→木
  （木生火、火生土、土生金、金生水、水生木）。区域**生**你的符 → ×1.3。
- **相克（符 ↔ 怪物，决定对位）**：木→土→水→火→金→木
  （木克土、土克水、水克火、火克金、金克木）。你的符**克**怪 → ×1.5；被怪克 → ×0.67。

> 一句话记忆：**相生用来摆位（增伤地基），相克用来选打谁（针对克制）。** 合成元素（雷/冰/毒/幽/阴/阳等）的克制关系见第六部分配方表。

## 2.2 地图空间模型（v2 缺失，先钉死）★

v1 空间模型（可搭原型、可真机改）：

- 场地中央/底部是**护道阵眼**（= 你要守的目标，血量 = 你的 HP 三围）。
- 怪物从场地上方/边缘**分波推进**，沿本局地图路径走向阵眼。
- **★修正：地图路径不是固定一条。** 每关从路径模板池中抽 1 个地图样式；首测至少 3 套：直进压迫、折线路径、双入口汇流。后续章节继续增加路线机关和多入口变化。
- **★修正：开放格位与格位属性都随机。** 每张地图有 8–10 个候选区域格，本局随机开放 6 个；开放出来的 6 个格再各自随机五行。玩家把 5 张符放进其中 5 个。
- 怪被清掉 = 没事；**怪走到阵眼 → 扣阵眼血**（按 2.x 承伤公式）。
- **阵眼血归零、且复活用尽 = 失败**。

> 即"护道阵眼 + 随机路径 + 随机开放格位"的塔防骨架。这样每局不是只换五行，而是**路径压力、可用空间、格位五行**一起变化，避免玩家背固定站位。

## 2.3 胜负条件（v2 没写明，先钉死）★

- **胜：** 打完该关全部 **7 波** → 通关，领奖、解锁扫荡。
- **负：** 护道阵眼血量归零 **且本局可用复活次数用尽** → 结算失败。
- 阵眼血 = HP 三围；减伤 = DEF 三围（公式见 4.4）。这样三围**全部**在局内有明确作用：ATK→输出、HP→阵眼血、DEF→减伤。

## 2.4 一局三段（锁死 1–3 分钟）

| 段 | 时长 | 玩家做什么 |
|---|---|---|
| ① 准备 | 15 秒 | 查看本局路径 + 随机开放的 6 个区域格，把 5 张符放进其中 5 个，凑相生并看射程 |
| ② 波次战斗 | 90–120 秒 | 自动战斗 + 每隔几波升级/融合 + 可选凌空画符 |
| ③ 结算 | 瞬间 | 通关领奖 / 解锁扫荡；阵眼破按 2.10 规则复活 |

## 2.5 技能槽 + 区域摆放（策略核心）

- 上场固定 **5 格符**，局外配好带进来；每张符自带 1 个五行；**五行配置局外定，进局不变**。
- 地图先从候选点中**随机开放 6 个区域格**，每个开放格再**随机一个五行** → 你只有 5 张符，**得挑 6 格里相生与射程综合最好的 5 个来放，主动弃掉最差的 1 格**。
- 判定（区域五行 **生** 符五行）：

| 关系 | 系数 | 例 |
|---|---|---|
| 相生（区域生符） | **×1.3** | 木区放火符（木生火） |
| 中性 | ×1.0 | 不相干 |
| 错配（区域克符） | **×0.8** | 水区放火符（水克火） |

> 路径、开放格位、格位五行都随机 → 每局最优摆法不同，这是耐玩点，也是"动脑也能强"的免费玩家爽点。深度三来源：① 地图与格位随机 ② 局内升级/融合 ③ 局外换带哪 5 个。

## 2.6 波次表（第 1 关示例）

一关 **7 波**，每波间隔 **10 秒**，总战斗 ~90 秒。

| 波 | 怪物 | 设计意图 |
|---|---|---|
| 1–2 | 单一五行弱怪 | 热身，让玩家看清自己克谁 |
| 3–4 | 混合两种五行 | 逼玩家不能只靠单属性 |
| 5 | 精英怪（带护盾，需克制破盾） | 小高潮，考验克制 |
| 6 | 混合 + 加速 | 节奏拉紧 |
| 7 | 冲锋波（数量压制）/ 章末为 Boss | 收尾爽点 |

> **护盾机制（补充）：** 精英怪护盾 = 一层额外血；用**克制它的属性**打，护盾受满伤；用**被它克的属性**打，对护盾伤害大幅衰减（破盾极慢）。这让"克制"在精英波成为硬需求，而非可有可无。

## 2.7 画符识别机制（v2 含糊，先钉死）★

- **基础档：** 不需要画，符自动施放达基准效果。摸鱼/手残零损失、零惩罚。
- **进阶档（可选）：** 战斗中用手指**描画该符的符文笔迹**，系统用模板匹配算**吻合度 0~100%**：

| 吻合度 | 效果 |
|---|---|
| ≥ 80% | 满额 **+20%** 威力 + 隐藏特效 |
| 50%–80% | 线性部分加成（如 +10%），无特效 |
| < 50% | 视同基础施放，**无加成也无惩罚** |

- 每张符的符文图案**固定**，局外可预览/练习；容差刻意放大，性能上用轻量笔迹识别（方向序列/$1 类算法），小游戏可承受。
- 设计意图：**手好的人多榨 30%，手残永远不亏。** 是上限不是门槛。

## 2.8 局内成长

# 第三部分 · 修为境界（等级机制 · 不封顶）

## 3.1 一句话

- **修为境界 = 主等级**，贯穿全程，决定"能玩到什么、基础多强"。
- **怎么涨：** 打关卡/副本/秘境得**道行经验**填满当前「层」；每境界满 **9 层**后**突破（渡劫）**进下一境界。
- **为什么不封顶：** 修仙天然无尽——人间九境后飞升仙界、仙界后开天；关卡/秘境推荐战力随境界同步上涨，**比的是相对值不是绝对值**，所以不封顶也不崩（见 3.6）。

## 3.2 境界阶梯（不封顶结构）

| 段 | 境界顺序 | 层 | 定位 |
|---|---|---|---|
| 人间九境 | 练气→筑基→金丹→元婴→化神→炼虚→合体→大乘→渡劫 | 每境 9 层 | 新手→成长→中期主体（约 81 级体量） |
| 仙界 | 真仙→金仙→太乙→大罗→道祖 | 每境 9 层 | 后期，配合大版本放出 |
| 无尽 | 开天 N 重 / 鸿蒙 N 阶（纯数字递增） | 无限 | 长线肝，永不封顶 |

- **每升 1 层**：发固定**属性点**（自由分配进三围 / 五行修为）+ 少量基础三围（具体数见第六部分）。
- **每突破 1 境界**：基础三围**大幅跃升** + **解锁新功能**（见 3.3）+ 消耗突破材料。

## 3.3 ★「什么境界能干什么」功能解锁表

把"等级"和"能玩到什么"绑死。新功能跟着境界**逐步解锁**——不淹没新手，又让爬境界一路有新东西拿。

| 境界 | 解锁功能 / 此时能做的事 |
|---|---|
| **练气 1** | 核心循环：5 符出战、6 区域相生摆放、自动战斗、灵机点升级、第 1 章 |
| **练气 3** | **画符进阶档**（手绘 +20%） |
| **练气 5** | **五行精融合**（先解锁雷/冰/毒，后续开放幽/阴/阳等合成元素技） |
| **练气 7** | **法器**系统（主攻击养成线） |
| **筑基 1** | **扫荡**（已通关关卡一键产出） |
| **筑基 3** | **道袍**系统 + 每日材料副本 |
| **筑基 6** | **首饰**系统（词条组合） |
| **金丹 1** | **灵兽**系统 |
| **金丹 3** | **PvP 段位赛** |
| **金丹 6** | **灵宝**系统 |
| **元婴 1** | **无尽秘境（符塔）**开放——无限爬塔（见 3.7） |
| **元婴 3** | **形象 / 外观**系统 |
| **元婴 +** | 宗门 / 帮派、世界 Boss、跨服（后续版本逐步放） |
| **化神 +** | 第 2~N 章主线、更高阶副本、新合成元素 |
| **仙界各境** | 仙阶专属玩法 / 装备品阶（大版本载体） |

> 解锁节奏原则：**前 3 个境界把"会影响一局打法"的功能开完**（画符、融合、法器、扫荡、道袍、首饰），社交/无尽/外观往后放。

## 3.4 突破（渡劫）+ 失败保护 ★

- **触发：** 当前境界 9 层满。
- **操作：** 消耗**渡劫丹 + 突破材料**（突破副本产），点"渡劫"；打一场**渡劫挑战战**（对应强度关卡）。
- **成功：** 进下一境界，三围大跳 + 解锁该境界功能。
- **失败保护（v2 缺）：** 挑战失败**不掉级、不损材料**，可冷却后重试；**连续失败 3 次 → 触发保底**（降低挑战强度或给临时增益），杜绝硬卡死。
- **卡瓶颈：** 材料不够 → 引导去刷/(后期)氪，但**不强制即时**。这是天然付费点，v1 阶段靠时间 + 广告即可过。
- **意义：** 把线性升级切成"一段一段有目标感"的里程碑节奏。

**★新增：突破节奏表（首测默认采用）**

| 突破 | 材料需求 | 免费预计 | 广告拉满预计 | 失败保护 |
|---|---|---:|---:|---|
| 练气→筑基 | 渡劫丹 10、筑基精华 5 | 2 天 | 1 天 | 连败 3 次后，挑战怪物 HP -15%。 |
| 筑基→金丹 | 渡劫丹 25、金丹精华 8 | 4 天 | 2–3 天 | 连败 3 次后，开局送 1 灵机点。 |
| 金丹→元婴 | 渡劫丹 50、元婴精华 12 | 7 天 | 4–5 天 | 连败 3 次后，Boss 护盾 -20%。 |

> 免费预计按第七部分的每日副本次数计算；广告拉满按每天额外翻倍 3–5 次关键副本计算。

## 3.5 五行修为 与 圆满（归位到境界之下）

- **五行修为** = 境界之下的 **5 条专精线**（金木水火土）。投属性点 → 长对应属性攻击 + 走克制专精。是战力权重里占比最高（30%）的"地基"。
- **圆满**：把某系修为推到**当前境界允许的上限** → 对"被它克的那种属性"克制 1.5 → **2.0** 且免疫"被克"减伤。标定：让专精玩家在该克制对局吃掉对手 ≤40% 战力差。
- **关系：境界决定上限，五行修为决定你怎么分配。** 境界涨 → 圆满上限跟着涨 → 圆满本身也**不封顶、可反复追**。

**★修正：PvP 单独平衡圆满克制**

| 场景 | 普通克制 | 圆满克制 | 被克处理 |
|---|---:|---:|---|
| PvE | 1.5 | 2.0 | 圆满后免疫被克减伤 |
| PvP | 1.5 | 1.7 | 圆满后被克从 0.67 提到 0.85，不完全免疫 |

> 这样 PvE 保留专精爽感，PvP 不会变成单属性猜拳一把定胜负。

## 3.6 不封顶为什么不崩（设计说明）

四件套保证"等级无限"≠"数值失控"：

1. **相对而非绝对**：关卡/秘境/PvP 推荐战力随境界同步上调。
2. **PvP 靠段位匹配**，不是绝对战力——低境界在自己段位照样有对手。
3. **越级有活路**：克制 + 相生 + 画功最多补 **~25–30%** 战力差（上限），保证免费/技术玩家不被纯数值碾死，又不无限大破坏养成动机。
4. **突破做闸**：渡劫材料/挑战卡住节奏，避免一天冲顶。

> 结论：**不封顶 = 永远有目标 = 长留存**；靠"相对战力 + 段位匹配 + 越级补差 + 突破闸"四件套，数值不会失控。

## 3.7 无尽秘境 / 符塔爬塔（给无限等级的无限内容）

- **无限的等级，需要无限的内容去花。** 这就是「符塔」之名的落地：一座**无尽符塔**，每层一个波次防守关，越爬越难。
- 每层推荐战力随层数**指数上涨**（公式见 6.7），**爬到打不动为止**；按到达层数发**周榜奖励** + 道行经验。
- 高境界主战场，也是 build/克制/画功的终极考场。主线给"解锁与剧情"，符塔给"无限挑战与排名"。

# 第四部分 · 战力设定（落地）

> 桥接：**境界提供"基础三围平台"**（随等级永远上涨、不封顶）；下面 8 条养成线决定**同境界玩家之间**堆出多少战力差。境界 = 共同上涨的地板，养成线 = 地板之上的高低。

## 4.1 三围与战力公式

- **攻击 ATK**：技能输出来源
- **生命 HP**：护道阵眼血量，归零即危
- **防御 DEF**：阵眼承伤减伤

```
战力 = 攻击 ATK × 6  +  生命 HP × 0.6  +  防御 DEF × 3
```
例：ATK 300 / HP 3000 / DEF 150 → 战力 = 1800 + 1800 + 450 = **4050**

**★新增：战力口径统一**

| 口径 | 包含内容 | 用途 |
|---|---|---|
| 基础三围战力 | 境界给的 ATK/HP/DEF | 表示等级地板，用来看同境界最低强度。 |
| 期望总战力 | 基础三围 + 五行修为 + 符等级 + 装备 + 其他养成 | 用于关卡推荐战力、卡关判断、匹配。 |
| 局内实时战力 | 期望总战力 + 局内升级/融合/临时增益 | 只在本局中生效，用于战斗调试。 |

> 后续表格凡涉及推关，默认使用**期望总战力**；凡涉及境界成长，默认标为**基础三围战力**。

## 4.2 养成线权重

| 养成线 | 战力占比 | 备注 |
|---|---|---|
| 五行修为（等级+属性点） | 30% | 地基，任何阶段生效 |
| 法器 | 20% | 主攻击，门派专属可做外观/特效/获取效率，强度不做付费独占 |
| 技能配置（5 符等级） | 13% | |
| 道袍 | 12% | 本体属性 + 品质倍率 |
| 首饰 | 10% | 词条组合 |
| 灵兽 | 8% | |
| 灵宝 | 5% | |
| 形象 | 2% | 可肝可氪，少量加成 |

## 4.3 输出公式：局内单次伤害

```
单次伤害 = 技能基础攻击 × 品质倍率 × 相生系数 × 克制系数 × (1 + 画功/特效加成)

其中  技能基础攻击 = 技能等级攻击 + 对应五行修为攻击 + 法器攻击
```

| 系数 | 取值 |
|---|---|
| **品质倍率** | 下品 1.0 / 中品 1.3 / 上品 1.7 / 极品 2.2 / 极品+ 3.0 |
| **相生系数**（区域↔符） | 相生 1.3 / 中性 1.0 / 错配 0.8 |
| **克制系数**（符↔怪） | 克制 1.5 / 中性 1.0 / 被克 0.67 |
| **画功/特效加成** | 基础 +0；进阶画好 +20%；隐藏特效另算 |

> **怪物防御：** v1 阶段怪物**只有 HP、无 DEF**（输出公式不减怪防），只有精英怪有"护盾"层（见 2.6）。怪物 DEF 留到中后期再加，避免初期数值复杂化。

## 4.4 承伤公式：阵眼怎么掉血（v2 完全缺失，先钉死）★

怪物突破到护道阵眼时：

```
阵眼受到伤害 = 怪物攻击 × (1 − DEF 减伤率)
DEF 减伤率 = DEF / (DEF + 600)        ← 600 为 v1 减伤常数，真机校准
```

例：DEF 150 → 减伤率 = 150 / 750 = **20%**；怪攻 100 → 阵眼掉 **80** 血。
**阵眼血（=HP 三围）归零 → 失败。** 这样 HP/DEF 在局内有了确定作用，和输出侧咬合成完整闭环。

## 4.5 圆满克制

把一系修为练到**圆满** → 对**被它克的那种属性**，克制 1.5 → **2.0** 且免"被克"减伤。只对"被你克的那一种"生效；混合关卡/PvP 仍需全面发展。

## 4.6 关卡推荐战力曲线（第 1 章，v1）

| 关卡 | 推荐战力 |
|---|---|
| 1-1 | 2000 |
| 1-2 | 2400 |
| 1-3 | 2900 |
| 1-4 | 3500 |
| 1-5 | 4200 |
| 章末 Boss | 6300 |

判定：战力 ≥ 推荐 → 稳过；< 推荐 80% → 大概率卡关 → 引导养成/氪；**技巧补差**：克制+相生+画功最多补 **~25–30%**。曲线如何无限延伸到后续章节，见 6.6。

# 第五部分 · 完整算例（证明咬合，输出 + 生存两侧）

## 5.1 输出侧（同一张符的 3 倍差）

玩家在 **1-3 关（推荐 2900）**，放火系「焚天符」：

1. 基础攻击 = 符等级攻 120 + 火修为攻 80 + 法器攻 100 = **300**
2. 上品 → ×1.7 → **510**
3. 放进木区（木生火，相生）→ ×1.3 → **663**
4. 打不同怪：火怪（中性）**663** / 金怪（火克金）**995** / 水怪（被克）**444**
5. 火修为圆满，打金怪（升 2.0）→ 510 × 1.3 × 2.0 = **1326**
6. 进阶画功打好（+20%）打金怪 → 995 × 1.2 = **1194**

> 同一张符，靠**摆位 + 对位 + 画功**，输出 444 → 1326，**三倍差**：数值定下限，操作定上限。

## 5.2 生存侧（HP/DEF 怎么救命）★

阵眼 HP 3000、DEF 150（减伤 20%）。第 7 波漏进 5 只怪、每只攻 100：

- 每只对阵眼造成 100 ×（1−20%）= **80**；5 只共 **400** → 阵眼剩 2600，守住。
- 若把属性点全堆 ATK、DEF 只 30（减伤 30/630 ≈ 5%）：每只 95、5 只 **475**，且因输出不足漏更多 → 容易破阵。

> 结论：**ATK 决定你清不清得掉，HP/DEF 决定漏几个还能不能扛**。两侧都要，逼出"分配"决策。

## 7.2 体力 / 次数模型

- **主线推图：不耗体**，随便打，卡你的是战力不是次数（保证"想玩就能玩"）。
- **副本 / 秘境 / 突破本：耗"次数"或体力**（每日限），用来**控制养成材料产出节奏**——这是长线留存与（后期）付费的主阀门。
- 体力随时间恢复；钻石/广告可买额外次数（v1 阶段靠广告）。

## 7.3 装备产线（材料 → 装备 → 战力）

```
打材料副本 → 攒分系材料 → 打造装备(法器/道袍/首饰) → 强化(+等级) → 升品(下→极品+) → 战力上涨
```
- 每个养成系（4.2 那 8 条）各有自己的材料本与产线，按 3.3 解锁表逐个开放，避免一次全糊脸。

## 8.2 PvP 细化

- **形态：** v1 做**异步**——打对手的**AI 镜像**（用其配置 + 战力跑一局攻防），不做实时同步（小游戏成本/体验考量）。
- **匹配：** 按**段位 + 战力**匹配，不是绝对战力 → 低境界在自己段位有对手、有胜负。
- **★修正：圆满克制在 PvP：** PvE 圆满克制保留 2.0；PvP 圆满克制降为 1.7，且被克时从 0.67 提到 0.85，不完全免疫。鼓励"专精 vs 全面"的博弈，不做一招鲜。
- **奖励：** 段位周结发钻石/材料/外观，纯荣誉 + 养成，不卖数值（合规友好）。

## 8.3 商业化框架（先合规、避版号，再开闸）

- **上线首版（无内购）：** 纯免费 + 广告变现——**复活、奖励翻倍、额外免费次数**看广告。无任何充值入口 → 规避版号前置要求。
- **拿到版号后再开内购：** 月卡、礼包、抽符、外观、体力/扫荡券。
- **关键：付费点早已埋在系统里**（突破材料、体力、扫荡、外观），后期"开闸"即可，**不需要回炉改设计，也不破坏前期免费体验**。
- 红线：**不卖一锤定音的数值碾压**；靠"越级补差 + 段位匹配"让免费玩家始终有活路，保口碑与长留存。

## 8.4 开发顺序（吸收旧"下一步"，更详）

1. **可玩原型**：1 关 7 波 + 5 符 6 格 + 自动战斗 + **输出公式(4.3) + 承伤公式(4.4)** + 练气~筑基境界框架 → 真机调 2.8 节拍、3.3 解锁、4.6 曲线，找咬合点。
2. **数值全量落地**：把第六部分所有表填实 + 第 1 章 5 关 + 推荐战力曲线校准。
3. **养成系统逐个接**：按 3.3 解锁表顺序（法器→道袍→首饰…）+ 各自材料副本 + 7.3 产线。
4. **长线内容**：无尽符塔(6.7) + PvP 段位(8.2) + 周榜。
5. **留存系统**：日常/任务/登录(7.4) + 广告变现。
6. **（拿到版号后）商业化**：按 8.3 开内购闸。
