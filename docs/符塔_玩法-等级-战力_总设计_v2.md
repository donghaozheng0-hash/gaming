# 符塔 · 玩法 + 等级 + 战力 · 总设计（落地版 v2）

> 本版在 v1 基础上补三块,目标是「外人扫一眼就懂这游戏怎么玩、怎么设计、什么境界能干什么」：
> ① **0 章 30 秒总览** ② **第一部分 完整操作清单** ③ **第三部分 不封顶的修为境界（等级）机制 + 功能解锁表**。
> v1 已经定死的核心循环（第二部分）与战力系数（第四部分）**原样保留**，只加桥接说明，不改数值。
> 原则不变：宁可先定死一个值，先跑原型，再真机调手感。

## 0 · 一眼看懂（30 秒讲清这游戏）

**一句话：**
《符塔》是一款玄幻「画符塔防 Roguelite」小游戏。你是一名修士，**带 5 张符出战**；把符摆进每局随机的五行阵位凑**相生**，靠**克制**选打谁，关键时刻**手绘画符**多榨威力，在 **1–3 分钟**的波次防守里越级斩妖。局外则**无限提升修为境界**、解锁新系统、爬无尽符塔。

**你每一局只做三件事（核心爽点三件套）：**

| 三件套 | 操作 | 收益 |
|---|---|---|
| **摆位** | 把符放进「相生」的区域 | 伤害 ×1.3 |
| **对位** | 用「克制」怪物的属性去打 | 伤害 ×1.5 |
| **画功** | 手绘画得好（可选） | 再 +20% / 加隐藏特效 |

> 一句话设计哲学：**战力打底（下限），技巧翻盘（上限）。** 同一张符，光靠这三件套，输出能差到 3 倍（见第五部分算例）。

**类型定位：自动战斗为主 + 可选手绘动作层。**
- 摸鱼党 / 手残：全自动也能正常推图，零损失。
- 想秀的人：手绘能多榨 ~30%，肝功能/技术能补一部分战力差。
- 差异化：刻意避开「塔防 + 挂机」红海——**别人挂机，你动脑 + 动手；但又不逼手残硬操作。**

# 第一部分 · 你能做哪些操作（操作清单）

把「玩家到底能点什么、拖什么、画什么」一次列清。分**局内**（一局之内）和**局外**（养成系统）。

## 1·A 局内操作（一局 1–3 分钟之内）

| 阶段 | 操作 | 怎么做 | 作用 |
|---|---|---|---|
| ① 准备（15s） | **拖拽布阵** | 把 5 张符拖进 6 个区域格 | 凑相生 ×1.3，是策略核心 |
| ① 准备 | **查看阵位五行** | 点区域看它本局的属性 + 相生/错配提示 | 决定怎么摆 |
| ② 战斗（90–120s） | **自动 / 点击施放** | 默认自动放符；也可点符手动放 | 输出 |
| ② 战斗 | **凌空画符**（可选） | 在屏幕上手绘符文 | 画好 +20% 或挂隐藏特效 |
| ② 战斗 | **召出成符** | 一键放出局外预存好的「成符」 | 手残的保底强招 |
| ② 战斗 | **用道行点** | 第 2/4/6 波各得 1 点，二选一：升级 or 融合 | 局内即时变强 |
| ③ 结算 | **领奖 / 扫荡 / 复活** | 通关领奖；已通关关卡可扫荡；失败可复活 ×3 | — |

## 1·B 局外操作（养成 / 系统）

| 系统 | 操作 | 作用 |
|---|---|---|
| **出战配置** | 选定 5 张符 + 各自五行 | 决定本局 build（深度不靠加槽，靠换带哪 5 个） |
| **修为境界** | 攒道行经验升「层」、满级「突破/渡劫」 | 涨等级、解功能、长基础三围（**不封顶**） |
| **五行修为** | 把属性点分配到金木水火土五条线 | 长对应属性攻击、走克制专精、冲「圆满」 |
| **装备养成** | 法器 / 道袍 / 首饰 / 灵兽 / 灵宝 / 形象 的强化与更换 | 堆战力（差异化主力） |
| **融合** | 用道行点 + 五行精解锁/强化组合技 | 解锁雷 / 冰 / 幽 等合成元素技 |
| **副本** | 刷材料本 | 产出养成材料 |
| **扫荡** | 一键扫已通关关卡 | 省时拿产出 |
| **PvP 段位赛** | 排位对战 | 段位、屠榜 |
| **无尽秘境（符塔）** | 一层层往上爬 | 无限挑战，消化高境界，冲周榜 |

# 第二部分 · 核心玩法（落地，承接 v1，数值不动）

## 2.1 一局三段（锁死 1–3 分钟）

| 段 | 时长 | 玩家做什么 |
|---|---|---|
| ① 准备 | 15 秒 | 把 5 张符放进随机五行区域，凑相生 |
| ② 波次战斗 | 90–120 秒 | 自动战斗 + 每隔几波升级技能 + 可选凌空画符 |
| ③ 结算 | 瞬间 | 通关领奖 / 解锁扫荡；失败可复活 3 次 |

## 2.2 技能槽（5 格，种子 build）

- 上场固定 **5 格**，局外配好带进来；每格 = 1 张符，自带 1 个五行。
- 开局可「五个全火」也可均衡，**五行配置局外定，进局不变**。
- 深度三来源：① 局内升级 ② 五行融合解锁组合技 ③ 局外换带哪 5 个。

## 2.3 区域 + 相生摆放（每局随机，策略核心）

- 防守地图默认解锁 **6 个区域格**，每格**本局随机**一个五行（后期掺合成元素）。
- 把符放进区域，按「区域五行 **生** 符五行」判定：

| 关系 | 系数 | 例 |
|---|---|---|
| 相生（区域生符） | **×1.3** | 木区放火符（木生火） |
| 中性 | ×1.0 | 不相干 |
| 错配（区域克符） | **×0.8** | 水区放火符（水克火） |

> 区域每局随机 → 每局最优摆法都不同，这就是耐玩点，也是「动脑也能强」的免费玩家爽点。

## 2.4 波次表（第 1 关示例，v1）

一关 **7 波**，每波间隔 **10 秒**，总战斗 ~90 秒。

| 波 | 怪物 | 设计意图 |
|---|---|---|
| 1–2 | 单一五行弱怪 | 热身，让玩家看清自己克谁 |
| 3–4 | 混合两种五行 | 逼玩家不能只靠单属性 |
| 5 | 精英怪（带护盾，需克制破盾） | 小高潮，考验克制 |
| 6 | 混合 + 加速 | 节奏拉紧 |
| 7 | 冲锋波（数量压制）/ 章末为 Boss | 收尾爽点 |

## 2.5 局内成长（v1 机制，落地）

- **道行点**：第 2、4、6 波结束各给 1 点 → **每局 3 点**。
- 用途（二选一）：**升级一张符**（+15% 该符攻击/级）或 **融合**（消耗道行点 + 五行精，解锁/强化组合技）。
- **五行精**：每波 30% 概率掉 1 颗，用于融合解锁合成元素技（雷/冰/幽等）。
- **平衡铁律**：玩家成长曲线**略低于**怪物强度曲线，做出「勉强守住、偶尔告急」。3 个点正好让玩家每两波做一次「升哪个」的小决策——给太多就无伤过关（无聊），太少就守不住（劝退）。

## 2.6 画符（两档，摸鱼党不被挡）

- **基础档**：符可一键/自动施放，达基准效果；预画的符存「成符」一键召出。摸鱼/手残零损失。
- **进阶档（可选）**：实时手绘画得好 → 该次施放 **+20% 威力 或 多挂 1 个隐藏特效**。锦上添花，非门槛。

## 2.7 失败与复活

每局 **3 次复活**（看广告 30→60 秒 / 或钻石）；超出则结算失败。

# 第三部分 · 修为境界（等级机制 · 不封顶）★ 本版新增核心

## 3.1 一句话：等级是什么、怎么涨、为什么不封顶

- **修为境界 = 你的主等级**，贯穿全程的成长主轴，决定你「能玩到什么、基础有多强」。
- **怎么涨**：打关卡 / 副本 / 秘境得**道行经验**，填满当前「层」；每个境界满 **9 层**后**突破（渡劫）**进下一境界。
- **为什么不封顶**：玄幻修仙天然无尽——人间九境之后飞升仙界，仙界之后开天，**境界阶梯永远向上**。配合关卡/秘境推荐战力随境界同步上涨：**你强、敌人也强，比的是相对值不是绝对值，所以不封顶也不崩**（详见 3.6）。

## 3.2 境界阶梯（不封顶结构）

分三段：前段用修仙命名给仪式感，后段用程序化数字命名以支持**真·无限延伸**。

| 段 | 境界顺序 | 层 | 定位 |
|---|---|---|---|
| 人间九境 | 练气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫 | 每境 9 层 | 新手 → 成长 → 中期主体（约 81 级体量） |
| 仙界 | 真仙 → 金仙 → 太乙 → 大罗 → 道祖 | 每境 9 层 | 后期，配合大版本放出 |
| 无尽 | 开天 N 重 / 鸿蒙 N 阶（纯数字递增） | 无限 | 长线肝，永不封顶 |

- **每升 1 层**：发固定 **属性点**（玩家自由分配进三围 / 五行修为）+ 少量基础三围。
- **每突破 1 境界**：基础三围**大幅跃升** + **解锁新功能**（见 3.3）+ 消耗突破材料（渡劫）。

## 3.3 ★「什么境界能干什么」功能解锁表

这是把「等级」和「能玩到什么」绑死的核心表。新功能跟着境界**逐步解锁**——既不淹没新手（渐进式开放），又让爬境界一路有新东西拿。

| 境界 | 解锁功能 / 此时能做的事 |
|---|---|
| **练气 1** | 核心循环开启：5 符出战、6 区域相生摆放、自动战斗、道行点升级、第 1 章 |
| **练气 3** | **画符进阶档**（手绘 +20%） |
| **练气 5** | **五行精融合**（解锁雷 / 冰 / 幽等合成元素技） |
| **练气 7** | **法器**系统（主攻击养成线开启） |
| **筑基 1** | **扫荡**（已通关关卡一键产出） |
| **筑基 3** | **道袍**系统 + 每日材料副本 |
| **筑基 6** | **首饰**系统（词条组合） |
| **金丹 1** | **灵兽**系统 |
| **金丹 3** | **PvP 段位赛** |
| **金丹 6** | **灵宝**系统 |
| **元婴 1** | **无尽秘境（符塔）开放**——无限爬塔（见 3.7） |
| **元婴 3** | **形象 / 外观**系统 |
| **元婴 +** | 宗门 / 帮派、世界 Boss、跨服（后续版本逐步放） |
| **化神 +** | 第 2 章…第 N 章主线、更高阶副本、新合成元素 |
| **仙界各境** | 仙阶专属玩法 / 装备品阶（大版本更新载体） |

> 解锁节奏原则（数字可在原型期微调）：**前 3 个境界把「会影响一局打法」的功能开完**（画符、融合、法器、扫荡、道袍、首饰），**社交 / 无尽 / 外观往后放**。新手别一上来面对 10 个系统。

## 3.4 突破（渡劫）—— 一个关键操作

- **触发**：当前境界 9 层满。
- **操作**：消耗**突破丹 / 渡劫材料**（副本产出），点「渡劫」；可选打一场**渡劫挑战战**（一关对应强度的关卡）。
- **成功**：进入下一境界，基础三围大跳 + 解锁该境界功能。
- **卡瓶颈**：材料不够 / 挑战没过 → 停在原境界继续肝或补战力。这是给肝 / 氪的**天然付费点**，但不强制（v1 阶段无内购，靠时间和广告即可过）。
- **意义**：把线性升级切成「一段一段、有目标感」的节奏，每个境界都是一个里程碑，而不是数字麻木上涨。

## 3.5 五行修为 与 圆满（承接 v1，归位到境界之下）

- **五行修为** = 修为境界之下的 **5 条专精线**（金 / 木 / 水 / 火 / 土）。把属性点投进去 → 长对应属性攻击 + 走克制专精。它仍是 v1 战力权重里占比最高（30%）的养成「地基」。
- **圆满**：把某一系五行修为推到**当前境界允许的上限** → 对「被它克的那种属性」克制系数 1.5 → **2.0**，且免疫「被克」减伤（v1 定值）。标定目标：让专精玩家在该克制对局中吃掉对手 ≤40% 的战力差。
- **关系**：**境界决定上限，五行修为决定你怎么分配。** 境界涨 → 圆满的上限也跟着涨 → 所以「圆满」本身也是**不封顶、可反复追**的目标。

## 3.6 不封顶为什么不崩（设计说明，给你定心）

四件套保证「等级无限」≠「数值失控」：

1. **相对而非绝对**：关卡 / 秘境 / PvP 推荐战力随境界同步上调，你和敌人一起涨。
2. **PvP 靠段位匹配**，不是绝对战力——低境界在自己段位里照样有对手、有胜负。
3. **越级有活路**：克制 + 相生 + 画功最多补 **~25–30%** 战力差（v1 上限），保证技术 / 免费玩家不被纯数值碾死，又不让它无限大破坏养成动机。
4. **突破做闸**：渡劫材料 / 挑战卡住升级节奏，避免一天冲到天花板，保证长线留存。

> 结论：**不封顶 = 永远有目标 = 长留存**；靠「相对战力 + 段位匹配 + 越级补差 + 突破闸」四件套，数值不会失控。

## 3.7 ★ 无尽秘境 / 符塔爬塔（给无限等级的无限内容）

- **无限的等级，需要无限的内容去花**，否则升级失去意义。这就是「符塔」之名的落地：一座**无尽的符塔**，每层一个波次防守关，越爬越难。
- 每层推荐战力随层数**指数上涨**，**爬到打不动为止**；按到达层数发**周榜奖励** + 道行经验。
- 高境界玩家的主战场，也是检验你 build / 克制 / 画功的终极考场。
- 与主线互补：**主线给「解锁与剧情」，符塔给「无限挑战与排名」。**

# 第四部分 · 战力设定（落地，承接 v1，数值不动）

> 桥接说明：**修为境界提供「基础三围平台」**（随等级永远上涨、不封顶）；下面这 8 条养成线，决定**同境界玩家之间**能堆出多少战力差（差异化主力）。两者不冲突：境界 = 共同上涨的地板，养成线 = 地板之上的高低。

## 4.1 三围与战力公式

三个核心属性（MVP 先做这三个，暴击等后期加）：**攻击 ATK**（输出来源）/ **生命 HP**（本命·守护目标血量，被突破即危）/ **防御 DEF**（减伤）。

**战力公式（v1）：**
```
战力 = 攻击 ATK × 6  +  生命 HP × 0.6  +  防御 DEF × 3
```
例：ATK 300 / HP 3000 / DEF 150 → 战力 = 1800 + 1800 + 450 = **4050**

## 4.2 养成线权重（每条该贡献多少战力）

| 养成线 | 战力占比 | 备注 |
|---|---|---|
| 五行修为（等级 + 属性点） | 30% | 地基，任何阶段生效 |
| 法器 | 20% | 主攻击，门派专属仅氪 |
| 技能配置（5 符等级） | 13% | |
| 道袍 | 12% | 本体属性 + 品质倍率 |
| 首饰（30 级） | 10% | 词条组合 |
| 灵兽 | 8% | |
| 灵宝 | 5% | |
| 形象 | 2% | 可肝可氪，少量加成 |

## 4.3 玩法 × 战力的接口：局内单次伤害公式

```
单次伤害 = 技能基础攻击 × 品质倍率 × 相生系数 × 克制系数 × (1 + 画功/特效加成)

其中  技能基础攻击 = 技能等级攻击 + 对应五行修为攻击 + 法器攻击
```

| 系数 | 取值 |
|---|---|
| **品质倍率**（符/装备品质） | 下品 1.0 / 中品 1.3 / 上品 1.7 / 极品 2.2 / 极品+ 3.0 |
| **相生系数**（区域↔符，见 2.3） | 相生 1.3 / 中性 1.0 / 错配 0.8 |
| **克制系数**（符↔怪物五行） | 克制 1.5 / 中性 1.0 / 被克 0.67 |
| **画功/特效加成** | 基础档 +0；进阶档画好 +20%；隐藏特效另算 |

## 4.4 圆满克制（给肝帝的窄路，承接 3.5）

把一系五行修为练到**圆满** → 对**被它克制的那种属性**，克制系数 1.5 → **2.0** 且免疫「被克」减伤。只对「被你克的那一种」生效；混合属性关卡 / PvP 仍需全面发展。

## 4.5 关卡推荐战力曲线（第 1 章，v1）

| 关卡 | 推荐战力 |
|---|---|
| 1-1 | 2000 |
| 1-2 | 2400 |
| 1-3 | 2900 |
| 1-4 | 3500 |
| 1-5 | 4200 |
| 章末 Boss | 6300 |

判定规则：战力 ≥ 推荐 → 可稳过；战力 < 推荐 80% → 大概率卡关 → 引导养成 / 氪金；**技巧补差**：克制 + 相生 + 画功最多补 **~25–30%** 战力差，这是「动脑/肝功能弥补数值」的上限。

# 第五部分 · 完整算例（证明它不虚，承接 v1）

玩家在 **1-3 关（推荐战力 2900）**，放一张火系「焚天符」：

1. 技能基础攻击 = 技能等级攻击 120 + 火修为攻击 80 + 法器攻击 100 = **300**
2. 符是**上品** → ×1.7 → **510**
3. 放进**木属性区域**（木生火，相生）→ ×1.3 → **663**
4. 打不同怪：
   - 打火怪（中性 ×1.0）→ **663**
   - 打金怪（火克金 ×1.5）→ **995**
   - 打水怪（水克火，被克 ×0.67）→ **444**
5. 若火修为**圆满**，打金怪（克制升 2.0）→ 510 × 1.3 × 2.0 = **1326**
6. 进阶档画功打好（+20%）打金怪 → 995 × 1.2 = **1194**

> 同一张符，因为**摆位（相生）+ 对位（克制）+ 画功**，输出从 444 跳到 1326——**三倍差**。这就是「战力打底、技巧翻盘」的咬合：数值决定下限，操作决定上限。

# 第七部分 · 研发落地修复意见（追加）

> 本章不是推翻前文，而是把当前版本里「研发会追问、数值会卡住、玩家可能误解」的地方先标出来。优先级按原型落地风险排序：先补规则，再补数值，再补长线。

## 7.1 优先修复项

| 优先级 | 问题 | 风险 | 修复建议 |
|---|---|---|---|
| P0 | **塔防底层规则还不完整** | 目前写清了 5 符、6 区域、7 波怪，但还没写怪怎么走、漏怪扣什么、符怎么攻击、射程怎么算。研发无法拆战斗任务。 | 新增「战斗底层规则表」，先定路线、基地血、符攻击频率、目标选择、漏怪伤害、Boss/精英规则。 |
| P0 | **技巧补差与三倍伤害表述容易冲突** | 文档说技巧最多补 25–30% 战力差，但算例里单次伤害能差 3 倍，容易被认为数值不自洽。 | 区分「单次最优伤害差」和「整局平均补差」。单次可以 3 倍，整局受覆盖率、冷却、怪物混合、画符次数限制，最终补差控制在 25–30%。 |
| P0 | **画符系统技术边界未定** | 手绘识别是核心差异点，但如果一开始做复杂识别，研发成本和手感风险都高。 | MVP 采用「轨迹模板评分 + 宽容判定 + 冷却施放」。画得好给 +20%，画错不惩罚，只按基础档释放。 |
| P1 | **局内 Roguelite 变化不足** | 现在局内只有 3 个道行点，选择偏少，长期会像固定数值塔防。 | 波后增加「三选一临时增益」，围绕五行和符类型做 build，让每局都有局内成长差异。 |
| P1 | **道行点 / 道行经验命名混淆** | 一个是局内临时资源，一个是局外等级经验，玩家和研发都容易混。 | 局内资源改名为「灵机点」或「悟道点」；局外保留「道行经验」。 |
| P1 | **不封顶等级容易变成数值跑步机** | 如果玩家升级后只是敌人也变强，会感觉没有真正成长。 | 每个大境界至少给一个可感知变化：新符、新怪机制、新地图机关、新合成元素、新副本或新外观。 |
| P1 | **圆满克制在 PvP 中可能过强** | 2.0 克制 + 免疫被克会让 PvP 变成猜属性，压缩阵容空间。 | PvE 保留强爽感；PvP 单独设平衡系数，例如圆满克制只到 1.65–1.75，且不完全免疫被克。 |
| P2 | **复活 3 次可能破坏短局节奏** | 1–3 分钟一局，如果失败连续看 3 次广告，节奏和观感都偏重。 | MVP 改为每局 1 次广告复活 + 1 次道具/钻石复活；无尽符塔可限制或禁用复活。 |

## 7.2 结构补强项

| 模块 | 当前状态 | 补强方向 |
|---|---|---|
| **新手教学** | 已有功能解锁表，但缺少首局到首章的教学节奏。 | 把核心机制拆到前 6–8 关逐步教：自动战斗 → 摆位 → 克制 → 画符 → 局内升级 → 融合。 |

## 7.3 数值修复项

| 数值点 | 风险 | 建议 |
|---|---|---|
| **品质倍率** | 下品 1.0 到 极品+ 3.0 跨度大，再叠相生、克制、画符后，高品质会过度压制策略。 | MVP 先压低极品+倍率，或让高品质更多提供机制差异，不只堆纯倍率。 |
| **HP / DEF 战力价值** | 塔防失败条件未明确时，生命和防御容易变成虚战力。 | 明确 HP 是本命血/基地耐久，DEF 影响漏怪伤害减免或本命减伤。 |
| **五行精掉落** | 每波 30% 可能出现整局不掉，导致融合玩法空转。 | 改为固定节点保底 + 随机额外掉落，例如第 3/6 波保底 1 颗。 |
| **章末 Boss 战力跳跃** | 1-5 到 Boss 从 4200 到 6300，涨幅较大。 | 可以保留卡点，但要明确玩家通过哪条产出链补战力：扫荡、材料本、法器强化、符升级。 |
| **技巧补差上限** | 若相生、克制、画符覆盖率过高，免费技巧可能吞掉养成价值。 | 用冷却、怪物混合、区域随机和画符次数限制，将整局平均补差控制在 25–30%。 |

## 7.4 商业化修复项

- **避免把强度付费写得太直白**：原文「门派专属仅氪」如果放在主攻击法器上，会显得 P2W。建议改成「门派专属外观 / 特效 / 获取效率」，强度可以加速但不独占。
- **复活广告要轻**：短局游戏的广告点适合放在失败后一次、扫荡加速、材料翻倍，不建议连续 3 次复活。
- **核心策略必须免费可玩**：相生、克制、画符、基础融合必须免费完整开放。付费卖效率、外观、资源加速和便利性。
- **无尽符塔适合做周榜和轻付费消耗**：周榜奖励、挑战次数、扫荡层数、外观称号都可以承载长线，但不要让付费直接买排名。

## 8.1 战斗底层规则表

| 项 | MVP 定义 | 研发备注 |

| **局内资源** | 第 2/4/6 波给「灵机点」各 1 点。 | 原「道行点」建议改名，避免和局外道行经验混淆。 |

| **五行精** | 第 3/6 波保底掉 1 颗；其他波 30% 概率额外掉落。 | 保证融合系统不空转。 |
| **画符触发** | 战斗中点击符或画符按钮进入短暂画符状态，完成后释放强化版。 | 不建议长时间暂停；可做 0.5 秒慢动作。 |
| **画符判定** | 轨迹相似度 ≥ 70 分为成功，≥ 90 分触发完美。 | 成功 +20%；完美可触发隐藏特效。失败按基础档释放。 |
| **画符冷却** | 单符画符冷却 15 秒，全局画符冷却 5 秒。 | 控制整局技巧补差，防止无限画符。 |
| **复活规则** | 每局 1 次广告复活 + 1 次道具/钻石复活。 | 无尽符塔可只允许 1 次或禁用。 |
| **扫荡规则** | 三星通关后开放扫荡，消耗体力/扫荡券获得基础产出。 | 扫荡不掉局内随机增益，只掉关卡配置产出。 |

| **疾行妖** | 风格偏木/火 | 移速快，容易漏怪 | 水系减速、木系缠绕、前排区域覆盖。 | 1-7 起 | 首版没有风属性，先挂五行之一。 |
| **厚甲妖** | 土/金 | 高血量或高护盾 | 克制属性 + 破甲符 + 单体爆发。 | 1-5 起 | 用来验证单体符价值。 |
| **护盾妖** | 任意五行 | 护盾未破前减伤高 | 使用克制属性破盾，融合技更快。 | 1-6 起 | 护盾条要明显显示。 |
| **群聚小妖** | 任意五行 | 数量多，压路线 | AOE 符、范围减速、落岩类延迟爆发。 | 1-4 起 | 用来验证群攻符价值。 |
| **分裂妖** | 木/水 | 死后分裂成小怪 | AOE、持续伤害、后排补刀。 | 1-12 起 | MVP 可作为首章后段变化。 |
| **精英妖将** | 关卡主属性 | 血厚、带护盾、漏怪扣血高 | 克制破盾 + 画符爆发 + 升级主力符。 | 1-8 起 | 每关最多 1–2 只，避免节奏拖长。 |
| **章末 Boss** | 多阶段属性 | 阶段切换、召唤、冲锋或护盾 | 需要完整使用摆位、克制、升级、融合、画符。 | 1-20 | Boss 属性阶段要提前提示，避免玩家觉得被阴。 |

### 8.3.3 符与敌人的咬合原则

| 原则 | 说明 |
|---|---|
| **每种敌人至少有 1 个明显反制点** | 快速怪怕减速，护盾怪怕克制破盾，厚血怪怕单体爆发，群怪怕 AOE。 |
| **每张符至少有 1 个高光场景** | 不做纯下位替代。弱符也要在某类怪或某种区域里有用。 |
| **首章不追求全系统复杂** | 先让玩家学会 5 个核心：摆位、克制、画符、升级、融合。 |
| **属性克制服务于识别，不服务于记忆负担** | UI 必须明确提示「推荐用什么打」，让玩家靠理解做决策，不靠背表。 |
| **整局补差看覆盖率** | 单次伤害允许非常爽，但整局平均补差要靠冷却、怪物混合和区域随机控制在 25–30%。 |
