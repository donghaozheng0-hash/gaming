## 3 · 交付物(四个文件，精确接口)

### 3.1 `core.ts` — 承伤(系数走参数)
```ts
export function coreDamage({ atk, def, relK }: { atk: number; def: number; relK: number }): number
```
- 减伤率 = `def / (def + relK × atk)`；阵眼掉血 = `Math.round(atk × (1 − 减伤率))`。
- `relK` 由**调用方传入**(业务层从 `balance.defense.relK` 读后传)。本函数是纯数学，不读 config。
- 验证：`coreDamage({atk:100,def:150,relK:0.68})` → `31`。

### 3.2 `damage.ts` — 符伤害(系数走参数)
```ts
export function runeDamage({ base, qualityMul, xiangshengMul, kezhiMul, drawBonus }:
  { base: number; qualityMul: number; xiangshengMul: number; kezhiMul: number; drawBonus?: number }): number
```
- = `Math.round(base × qualityMul × xiangshengMul × kezhiMul × (1 + (drawBonus ?? 0)))`。
- 各倍率由调用方从 `balance.damageFormula.*` 读后传入。本函数纯数学。
- 验证：`base300 × 1.7 × 1.3 × {1→663, 1.5→995, 2.0→1326, 0.67→444}`。

### 3.3 `power.ts` — 战力(系数从 config 读)
```ts
export function combatPower({ atk, hp, def }: { atk: number; hp: number; def: number }): number
```
- = `atk×atkWeight + hp×hpWeight + def×defWeight`，**权重从 `loadGameConfig().balance.powerFormula` 读**(atkWeight/hpWeight/defWeight)。
- ⚠️ golden 调用 `combatPower({atk,hp,def})` **不传权重**，所以本函数**必须自己从 config 读权重**(不得把 6/0.6/3 写进代码)。
- 验证：`combatPower({atk:100,hp:1000,def:50})` → `1350`。

### 3.4 `elements.ts` — 五行克制(规则从 config 读)
```ts
export function relation(attacker: string, target: string): "advantage" | "neutral" | "disadvantage"
```
- 从 `loadGameConfig().balance.elements.kezhiCycle` 读克制环：
  - `kezhiCycle[attacker] === target` → `"advantage"`(我克它)
  - `kezhiCycle[target] === attacker` → `"disadvantage"`(被它克)
  - 否则 → `"neutral"`
- ⚠️ 克制环**从 config 读**，不得把五行关系硬写进代码。
- 验证：`relation("wood","earth")="advantage"`；`relation("wood","metal")="disadvantage"`；`relation("wood","water")="neutral"`。

## 4 · 关键设计(为何两类不同)
- `core`/`damage`：系数(relK、品质/相生/克制倍率)是**情境相关**的，由调用方按场景从 config 读后**作为参数传入** → 函数保持纯数学、可被 golden 直接喂参数测试。
- `power`/`elements`：权重与克制环是**全局固定规则**，golden 调用不传它们 → 函数**内部从 `loadGameConfig()` 读**。仍是"无裸数字"(从 config 读)、仍是纯函数(给定 config+输入则输出确定)。

可加一个内部 helper 缓存 `loadGameConfig().balance`，避免重复加载。
